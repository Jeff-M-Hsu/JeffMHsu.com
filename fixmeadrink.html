Fix Me A Drink

Date: August, 2019

Contributor: Jeffrey Hsu

Purpose: Personal Project

Stack: PHP | CSS | AWS EB/EC2

A website that consumes TheCocktailDB API to output recipes of cocktails. The user can request a random cocktail or search for recipes by name or by ingredient. This website also has a responsive layout that supports mobile and 2K/4K resolutions.

I built this website as my version of an "Intro to PHP". The "Hello World" tutorials that are all over the internet are not very suitable for learning new languages, building things on my own is much more stimulating and I find that I retain more knowledge this way.

Why don't I use something like React for a site like this? The intent was to start learning php and templating so I don't have to repeat the method I used to build my personal website. If I were to build something like this again, my stack of choice would likely be React, PHP, MySQL or some combination of frameworks based off of those.

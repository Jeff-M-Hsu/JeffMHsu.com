Currency Converter

Date: 2017

Contributor: Jeffrey Hsu

Purpose: Personal Project

Stack: Java | Swing | HttpClient

I created a desktop application to consume the currencylayer API. The currencylayer API holds records of both updated and historical currency exchange rates for over 168 currencies, including cryptocurrency such as Bitcoin. I wrote the program as a way of getting more practice with Java, Swing, and to learn about HttpClient, which provides a lightweight solution for sending requests to the API.

You can check it out on Github here.
